O que é HTML?

(Voltar)

Me tornar um desenvolvedor tem sido uma tarefa árdua, entretanto, bastante satisfatória, principalmente quando se está transitando entre carreiras. Nesse contexto, decidi que deveria começar por aquilo que de fato entendi como essencial: HTML. A sigla HTML significa "HiperText Markup Language" ou, em português, Linguagem de Marcação de Hipertexto. Funciona, essencialmente, como a base da web, permitindo a criação de websites, bem como a inserção de novos conteúdos. Atualmente, encontra-se na versão 5.

Onde fazer?

(Voltar)

Dá para escrever o texto base de um site em bloco de notas. Contudo, existem alguns programas que auxiliam neste processo de construção. Uso o Visual Code Studio por ser uma recomendação dos professores.

Qual o real poder?

(Voltar)

Acredito que o poder de fogo do HTML é bastante amplo, visto ter encontrado sites bastante elaborados. Recentemente, encontrei um canal no YouTube que ensina tutoriais de como fazer verdadeiras obras de arte utilizando o HTML. Espero que brevemente consiga fazer uso de todo arsenal que esta ferramenta disponibiliza.

Tags HTML: Os tijolos da sua página web

(Voltar)

Imagine que você está construindo uma casa. Para isso, você precisa de diferentes tipos de materiais de construção: tijolos, madeira, vidro, etc. Cada material tem uma função específica e juntos eles formam a estrutura da casa.

No mundo da programação web, as tags HTML são como esses tijolos. Elas são as estruturas básicas que você usa para construir uma página da internet. Cada tag tem um significado específico e informa ao navegador como exibir o conteúdo da sua página. Veja, abaixo, as tags usadas inicialmente no curso.

- html: Raiz de um documento HTML.

- head: Contém metadados sobre o documento, como título e links para arquivos CSS.

- body: Contém o conteúdo visível da página.

- p: Define um parágrafo.

- h1 a h6: Definem títulos de diferentes níveis.

- ul e ol: Definem listas não ordenadas e ordenadas, respectivamente.

- li: Define um item de lista

- a: Cria um hiperlink.
